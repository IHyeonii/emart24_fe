## KDT_Emart Project


#### React를 이용하여 emart24 사이트 구성
+ OS: Windows
+ Nodes v16.18.0

#### React
+ json-server --port 3001 --watch ./src/data/data.json
+ npm start

+ npm i react-router-dom
+ npm install axios
+ npm install recoil
+ npm install recoil-persist

#### 구현기능
+ 상품페이지: 상품리스트와 상세
+ 장바구니 추가, 삭제, 수량변경
+ 회원가입, 로그인, 로그아웃
